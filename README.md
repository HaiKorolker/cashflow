# מעקב הוצאות — Personal Cash Flow Tracker

אפליקציית PWA (Progressive Web App) למעקב אישי אחר הוצאות, הכנסות, הוראות קבע וחסכונות.  
כל הנתונים נשמרים **אצלך במכשיר בלבד** — ללא שרת, ללא הרשמה, ללא עלות.

---

## 📲 התקנה

### אנדרואיד (מומלץ)
1. פתח את [haikorolker.github.io/cashflow](https://haikorolker.github.io/cashflow/) בדפדפן **Chrome**
2. לחץ על תפריט 3 הנקודות ← **"הוסף למסך הבית"** / **"Install app"**
3. לחץ "הוסף" ← האפליקציה מופיעה כאייקון עצמאי

### iPhone / iPad
1. פתח את הקישור בדפדפן **Safari** (בלבד — Chrome ו-Firefox לא תומכים ב-PWA על iOS)
2. לחץ על כפתור השיתוף ← **"הוסף למסך הבית"**
3. גרסת iOS 16.4 ומעלה מומלצת

### מחשב (Windows / Mac)
1. פתח את הקישור בכרום
2. בשורת הכתובת: לחץ על אייקון ההתקנה (⊕) בצד ימין
3. לחץ "Install"

## 🗄️ איפה הנתונים נשמרים?

הנתונים נשמרים ב-**IndexedDB** — מסד נתונים מובנה בדפדפן, ישירות על המכשיר שלך.

**מה זה אומר בפועל:**
- ✅ אין שרת — אין מי שרואה את הנתונים שלך
- ✅ עובד ללא אינטרנט לאחר ההתקנה הראשונה
- ✅ אין הרשמה, סיסמה או חשבון
- ⚠️ אם תמחק את נתוני הדפדפן/האפליקציה — הנתונים יאבדו
- ⚠️ אין סנכרון אוטומטי בין מכשירים

**לכן: גבה את הנתונים שלך בקביעות** דרך הגדרות ← "יצא גיבוי JSON"

## ❓ שאלות נפוצות

**האם אפשר לשתף בין שני אנשים (זוג / שותפים)?**  
לא כרגע. כל התקנה עצמאית ומנותקת. אין אפשרות לסנכרן בין מכשירים שונים. כדי לאפשר שיתוף צריך שרת מרוחק — תכונה שאינה קיימת כעת.

**האם האפליקציה עובדת ללא אינטרנט?**  
כן. לאחר ההתקנה הראשונה, כל הקבצים שמורים במטמון ואפשר להשתמש באפליקציה ללא חיבור לרשת.

**האם הנתונים שלי בטוחים?**  
הנתונים נשארים אצלך בלבד, לא עוברים לשום שרת. הקוד פתוח לעיון (open source בריפוזיטורי זה).

**כמה מקום תופס האחסון?**  
כל עסקה שוקלת כמה בייטים. אפילו 10,000 עסקאות שנתיות לא יגיעו ל-5MB.

## 🛠️ טכנולוגיות

| רכיב | טכנולוגיה |
|---|---|
| UI | Bootstrap 5.3 RTL |
| גרפים | Chart.js |
| יצוא Excel | SheetJS (XLSX) |
| אחסון | IndexedDB (דרך custom DB wrapper) |
| Offline | Service Worker (Cache API) |
| התקנה | PWA Manifest |
